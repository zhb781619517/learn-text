import 	几种调用模块的方式
import module	 导入该module，在使用module的函数的时候，格式为module_name.func()
from module_name import function_name	导入module中的指定函数，该方法调用函数时，可以不加module_name
from module_name import function_name as fn	  导入module中指定函数，并将其命名为fn
import module_name as mn			导入module，并将其命名为mn
from module_name import *			导入module，并不命名，但是很危险，易与自己创造的函数形成命名冲突

技巧：
给函数或者变量命名时，要命名清晰明了，不要使用汉语拼音，尽量使用英文，善于利用下划线来表示关系，适当的使用小写和大写能解决很多问题
给形参指定默认值的时候，等号两边不要使用空格
调用的关键字实参也采用同样的规定
